# 元素週期表理化小遊戲

## 簡短描述

繁體中文元素週期表與理化小遊戲，結合元素資料、分類篩選、國中常見重點與隨機元素挑戰。

## 教育工作者摘要

元素週期表理化小遊戲是一個可直接在線上使用的繁體中文互動網站。首頁提供完整週期表，學生可切換國中常見元素、全部元素、金屬、類金屬、非金屬、鹼金屬族、鹼土金屬族、鹵素與鈍氣等篩選，點選元素後查看原子序、週期、族、分類、電子層、常見用途、生活連結與課本程度重點。網站也內建元素挑戰模式，包含簡單的元素符號與中文名稱配對、正常難度的國中理化常識題，以及答錯即結束並可登錄成績的無限挑戰。它適合用於國中自然科的元素符號、週期表分類、常見元素用途與化合物概念複習。

## 教學用途

- 在元素週期表單元中讓學生查找元素符號、中文名稱、週期、族與分類，建立查表習慣
- 作為課前暖身或課後複習，用五題元素挑戰檢核學生對常見元素與理化情境的理解
- 搭配金屬、非金屬、鹵素、鈍氣等篩選，引導學生比較同族元素與常見性質

## 導入注意

- 題庫與元素說明以國中理化概念複習為主，若用於正式評量，建議教師先檢查題目與課程版本是否一致
- 無限挑戰與英雄榜適合提高動機，但不宜作為唯一成績依據，仍需搭配學生解釋與紙筆或實作活動
- 網站包含外部 API 讀取訪客數與排行榜，若校園網路限制外部服務，相關功能可能無法顯示

## 啟動或安裝方式

直接開啟線上網站即可使用，不需要安裝軟體。學生可先在週期表頁面瀏覽或篩選元素，再點選「元素挑戰」進入簡單、正常或無限挑戰模式。
