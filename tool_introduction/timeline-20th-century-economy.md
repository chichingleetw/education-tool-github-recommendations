# 經濟轉折｜1900–現在

## 簡短描述

1900 到 2026 年全球與台灣經濟史互動時間軸，對照危機、制度、企業、科技、能源與人物。

## 教育工作者摘要

經濟轉折｜1900–現在是一套可部署到 GitHub Pages 的純前端互動經濟史教材，時間範圍涵蓋 1900 到 2026 年，並以亞洲、歐洲、美洲與台灣四條水平時間軌展示 71 則經濟史事件。學生可透過年份滑桿、拖曳、觸控板與方向鍵操作，勾選危機與衝擊、制度與政策、產業與科技、企業與市場、人物與思想、能源與環境等分類，觀察大蕭條、布雷頓森林體系、石油危機、亞洲金融風暴、金融海嘯、疫情、AI 革命、半導體、台灣十大建設、高鐵、碳費與代表企業等轉折。每張事件卡提供事件經過、因果與制度分析、關鍵人物、學術判讀、經濟名詞定義、官方或國際組織來源、YouTube 解說影片與 Wikimedia Commons 圖像，適合高中公民、歷史、地理與大學通識或經濟史課程。

## 教學用途

- 在高中公民、歷史、地理或大學通識課中比較全球與台灣經濟制度、產業、能源與企業發展的長期變化
- 讓學生依年份、區域與分類追蹤大蕭條、石油危機、金融風暴、疫情與 AI 革命等事件的因果脈絡
- 使用經濟名詞、學術判讀、資料來源、圖像與影片，引導學生練習制度分析與跨區域經濟史比較

## 導入注意

- 時間軸是教學型摘要，不宜把單一經濟事件視為只有一個原因，建議引導學生比較官方資料、統計與不同觀點
- 影片、圖像與外部來源可能更新、下架或禁止嵌入，正式授課前建議抽查連結與授權標示
- 大學程度名詞對高中生可能偏難，教師可先挑選少量事件與關鍵詞，再逐步增加制度與因果分析深度

## 啟動或安裝方式

可直接開啟 GitHub Pages 線上版本使用。若要本機測試，下載或 clone 專案後可直接用瀏覽器開啟 index.html；專案為純 HTML、CSS、JavaScript，不需安裝套件。也可在專案目錄執行 python3 -m http.server 8000，再開啟 http://localhost:8000。
