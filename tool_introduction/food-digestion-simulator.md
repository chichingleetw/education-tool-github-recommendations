# 消化道大冒險｜食物旅程模擬器

## 簡短描述

互動式食物消化過程模擬網頁，可選擇不同食物、份量與咀嚼程度，觀察 0 到 72 小時的消化旅程。

## 教育工作者摘要

消化道大冒險是一個可直接部署到 GitHub Pages 的純前端生物教學模擬器。學生可選擇水、飲料、米飯、牛排、蔬菜、珍珠奶茶、油炸食物等多種食物，調整份量與咀嚼程度後，用時間軸觀察食物在口腔、食道、胃、小腸、大腸、直腸到排出體外的移動與狀態變化。工具會即時顯示目前器官、食物狀態、消化程度、吸收比例與下一階段，也提供播放、暫停、倍速、階段跳轉、食物圖例與挑戰題，適合用來把「不同食物消化速度不同」「小腸是主要吸收場所」「液體與高脂肪食物通過胃部速度不同」等概念轉成可觀察的課堂活動。

## 教學用途

- 在自然科學、生物或健康教育課中說明消化道器官順序、食物移動與營養吸收位置
- 讓學生比較液體、澱粉、蛋白質、高纖維與高脂肪食物在胃、小腸與大腸中的停留差異
- 透過份量、咀嚼程度、時間軸與挑戰題，引導學生討論模型假設、飲食習慣與消化概念

## 導入注意

- 本工具的時間、消化程度與吸收比例是教學用簡化模型，不可作為醫療診斷或個人健康判斷
- 真實消化速度會受體質、食物份量、脂肪含量、咀嚼程度與健康狀況影響，教師應補充個體差異
- 食物分類與旅程以概念理解為主，若用於正式評量，建議搭配課本圖表、器官功能整理與學生解釋任務

## 啟動或安裝方式

可直接開啟 GitHub Pages 線上版本使用。若要本機測試，可下載或 clone 專案後直接用瀏覽器開啟 index.html；也可以在專案資料夾執行 python3 -m http.server 8000，再開啟 http://localhost:8000。
