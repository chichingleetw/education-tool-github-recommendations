# Classroom Timer 倒數計時器

## 簡短描述

大畫面倒數計時器，支援欄位滾輪、快速加減、最後秒數提醒、沙漏動畫與提示音。

## 教育工作者摘要

Classroom Timer 的倒數計時器模式適合需要清楚結束點的課堂活動，例如限時討論、測驗、實作任務、休息時間、展演準備或輪站切換。教師可以用滑鼠滾輪、上下按鈕、觸控滑動、鍵盤方向鍵與快速加減設定時分秒與百分秒，並可啟用最後 N 秒背光提醒、沙漏動畫、純文字模式與瀏覽器 Web Audio API 產生的提示音。工具不使用外部音訊檔、後端或網路 API。

## 教學用途

- 為討論、閱讀、測驗、實作或休息設定明確倒數時間
- 用最後秒數提醒與大畫面進度，協助學生提前收尾與轉換活動
- 在觸控螢幕或電子白板上快速調整時間，支援臨場延長或縮短任務

## 導入注意

- 瀏覽器可能因自動播放政策限制提示音，正式上課前應先測試聲音開關
- 倒數提醒能輔助節奏，但低年級或特殊需求學生仍可能需要口頭提醒
- 若裝置進入休眠、分頁被背景化或投影中斷，顯示效果可能受影響

## 啟動或安裝方式

可直接開啟 GitHub Pages 的倒數計時器連結。若要本機使用，可下載 classroom-timer-tzk 專案後直接開啟 index.html，或以靜態伺服器提供檔案；此模式可用 ?tool=countdown 直接進入。
