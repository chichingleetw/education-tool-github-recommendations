# Classroom Timer 即時時鐘

## 簡短描述

為教室投影與電子白板設計的大畫面即時時鐘，可顯示數位時間、日期與指針鐘。

## 教育工作者摘要

Classroom Timer 的即時時鐘模式是純前端教室時間顯示工具，適合在投影、電子白板或觸控螢幕上長時間顯示目前時間。它提供數位時間、日期、指針鐘、電子錶外框，以及多種字型、文字顏色與背景切換，方便教師依教室光線、投影設備與活動情境調整可讀性。工具不需要登入、後端或外部 API，偏好只留在目前瀏覽器中。

## 教學用途

- 在課堂、考試、分站活動或工作坊中投影目前時間，協助學生掌握節奏
- 搭配休息時間、集合時間或任務截止時間，提供全班一致的時間基準
- 在電子白板上切換指針鐘與數位時間，輔助低年級學生認讀時間

## 導入注意

- 即時時鐘依賴裝置本機時間，正式活動前應確認電腦或平板時間已校準
- 投影環境亮度不同，建議先測試背景與文字顏色，確保後排學生能清楚辨識
- 工具不會自動記錄學生行為或課堂事件，若需紀錄仍需搭配其他方式

## 啟動或安裝方式

可直接開啟 GitHub Pages 的即時時鐘連結。若要本機使用，可下載 classroom-timer-tzk 專案後直接開啟 index.html，或以靜態伺服器提供檔案；此模式可用 ?tool=clock 直接進入。
